# Style Monitoring Checklist

## Button Consistency
- [ ] All buttons use standardized size classes (btn-xs, btn-sm, btn-md, btn-lg)
- [ ] Button spacing follows form-actions rules
- [ ] Mobile responsive sizing works

## Form Elements
- [ ] Consistent border radius (8px)
- [ ] Proper spacing between form groups
- [ ] Uniform styling for inputs/selects

## Card Styling
- [ ] Consistent border radius (12px)
- [ ] Header background colors match
- [ ] Shadow levels uniform

## Testing Protocol
1. Review after any styling changes
2. Verify on multiple device sizes
3. Check color contrast
4. Test form validation states
